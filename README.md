# no-login-csdn-can-copy
不登录csdn也可以复制哦^o^


## 目录说明

- code                     为源码目录
- 移除某sdn复制限制v0.1.crx  可使用插件

## 安装 & 使用

1. 打开浏览器，在地址栏中粘贴chrome://extensions/并回车，勾选开发者模式
2. 将后缀为.crx的插件拖拽到浏览器中
3. 点击“添加”或“添加扩展程序”按钮。
4. 访问某sdn上的一篇文章或刷新网页，当页面上出现一句“可以复制啦^o^”时，表示可以正常复制。

### chrome浏览器

1. 打开浏览器，在地址栏中粘贴chrome://extensions/并回车，勾选开发者模式
2. 点击“加载已解压的扩展程序”，选择 **copy-csdn** 文件夹即可。
3. 点击“添加”或“添加扩展程序”按钮。
4. 访问某sdn上的一篇文章或刷新网页，当页面上出现一句“可以复制啦^o^”时，表示可以正常复制。
